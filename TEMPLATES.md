# Site templates and examples

This file contains optional templates for posts, tags/categories, and a footer snippet. Use them as a reference while keeping the site simple.

## Post template

```markdown
---
layout: post
title: "Post title here"
date: 2026-05-01 09:00:00 +0000
categories: notes
---

Write the post body in Markdown here. Keep sections short and use headings for structure.
```

## Tags / categories example

Jekyll supports categories and tags as part of front matter, but they are optional.

```yaml
categories: tooling
tags:
  - workflows
  - automation
```

A simple tag or category index can be added later if needed, but it is not required for a basic GitHub Pages blog.

## Footer snippet example

Use this example text for a footer or licensing note.

```html
<footer>
  <p>© 2026 Merp4. Content is licensed under a Creative Commons Attribution 4.0 International License unless otherwise noted.</p>
</footer>
```

You can copy this snippet into an HTML include or add it directly to your layout once you want a site-wide footer.
