# Getting started

This page provides an example introduction for using the site and publishing new content.

## Publishing new documentation

1. Create a new Markdown file in the `/docs` folder.
2. Add front matter with `title` and `permalink`.
3. Link the new page from `/docs/index.md`.

## Recommended structure

- Overview or purpose
- Step-by-step guidance
- Notes, commands, or configuration examples
- Related references
